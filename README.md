DNN 실시간 숫자 인식 도어락 시스템
TensorFlow/Keras의 DNN(Deep Neural Network) 모델을 활용하여 웹캠으로 입력되는 숫자를 실시간으로 인식하고, 설정된 4자리 비밀번호와 대조하는 보안 시뮬레이션 프로젝트입니다.

# 프로젝트 개요
기술적 목표: 딥러닝의 기초인 DNN 구조를 이해하고, 학습된 모델을 실제 라이브 영상 시스템(OpenCV)에 이식하는 파이프라인 구축.

핵심 기능:

웹캠 화면 중앙의 ROI(관심 영역) 내 숫자 실시간 추론.

4자리 비밀번호 순차 입력 및 일치 여부 판독 ( Welcome / GET OUT 메세지).

# 딥러닝 모델 설계 (DNN)
본 프로젝트는 MNIST 데이터를 활용하여 다음과 같은 심층 신경망 구조를 설계했습니다.

구조: 4개 층의 Dense 레이어 (256 - 128 - 56 - 10)

입력 전처리: 28x28 2D 이미지를 Flatten 레이어를 통해 1차원 벡터로 변환하여 입력.

활성화 함수: 은닉층은 ReLU, 출력층은 다중 분류를 위해 Softmax 사용.

학습 결과: 에폭(Epochs) 10회 반복 학습을 통해 검증 정확도 약 97.8% 달성.

# 영상 처리 파이프라인 (OpenCV)
딥러닝 모델이 실생활 노이즈가 섞인 웹캠 영상에서도 잘 작동하도록 강력한 전처리 과정을 설계했습니다.

Gaussian Blur: 웹캠 특유의 자잘한 노이즈를 제거하여 부드러운 이미지 생성.

Otsu's Thresholding: 주변 조명 환경에 상관없이 숫자를 배경과 명확히 분리하는 최적의 이진화 수행.

Morphological Erosion: 숫자의 획을 굵게 보정하여, 학습 데이터(MNIST)와 유사한 형태의 특징을 부각.

Normalization: 0~255의 픽셀 값을 모델 학습 규격인 0.0~1.0 사이로 정규화.
